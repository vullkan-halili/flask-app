# flask-app
Flask app api

## Prerequisites
`Mongodb`, `Python3` and `Virtualenv` should be installed.

## Installation
Install dependencies
``` text
bash install.sh
```

## Cofigure
Copy the `config/application-template.cfg` to `config/application` and add your settings.

## Run data extraction script.
Install dependencies
``` text
bash extract.sh
```

## Run flask server.
Install dependencies
``` text
bash run.sh
```
